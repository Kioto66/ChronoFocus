# worktracker-backend

Backend для отслеживания времени работы над задачами.
Точки входа:

**GET /summary?from=2025-01-01&to=2025-02-17**

_Параметры_

from - обязательный параметр, с какой даты начинать подсчёт (включительно)
to   - необязательный параметр, до какой даты считать (включительно), если не задан текущая дата.

_Ответ_

`{
"apiVer": "1",
"payload": [
    {
        "category": {
            "id": 1,
            "name": "Трудовая деятельность"
        },
        "time": "PT2H30M"
    },
    {
        "category": {
            "id": 1,
            "name": "Трудовая деятельность"
        },
        "time": "PT8H15M"
    }
],
"status": "OK"
}`

        "name": "Трудовая деятельность"

    }
  ],
  "status": "OK"
